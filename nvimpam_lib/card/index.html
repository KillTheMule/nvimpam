[−][src]Module nvimpam_lib::card

A datastructure to represent a Card of the FEM-Solver Pamcrash.

There is a 1-1 correspondence between a Card and a Keyword, but they are used differently. A Keyword is used as a parsing result for strings all over the place, while Card is used to define global static values the represent the concrete CARD entities of Pamcrash.

The definitions of the global static values can be found in the carddata module.
